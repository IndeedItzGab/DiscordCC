# Commands
These are all available commands that can used through Discord or in-game if specified.

## Discord
A slash commands available for Discord
### Moderator role(s) only
These slash commands are only available for moderators
|NAME                |PARAMETERS             |DESCRIPTION                                        |
|--------------------|-----------------------|---------------------------------------------------|
|announce            |\<context\>            |Send an announcement to the minecraft server.      |
|allowlist add       |\<gamertag\>           |Add a player to the whitelist.                     |
|allowlist remove    |\<gamertag\>           |Remove a pleyer from the whitelist.                |
|allowlist enable    |                       |Enable the whitelist on the server.                |
|allowlist disable   |                       |Disable the whitelist on the server.               |
|command             |\<command\>            |Run a command in the minecraft server.             |
|kick                |\<player\> [reason]    |Kick a specific player in the minecraft server .   |
|op                  |\<player\>             |Give a specific player a op permissions.           |
|deop                |\<player\>             |Remove the op permissions from the specific player.|
|server shutdown     |                       |Shutdown the minecraft server.                     |
|player location     |\<player\>             |Locate the coordinate of specified online player.  |

### Default
These slash commands are usable by anyone
|NAME          |PARAMETERS  |DESCRIPTION                                                       |
|--------------|------------|------------------------------------------------------------------|
|player list   |            |List all players online in the minecraft bedrock server.          |
|player info   |\<player\>  |View information about a specific player.                         |
|server info   |            |View information about the bedrock server.                        |
|host          |            |View the address and port of the server.                          |
|gamertag      |\<user\>    |Identify what gamertag is connected to the user's discord account.|
|register      |\<gamertag\>|Register your minecraft bedrock gamertag to connect to the server.|
