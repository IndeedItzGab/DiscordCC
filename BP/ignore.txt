DiscordCC V1.0.4
- Fixed an issue if the player was killed by a player doesn't show the killer's name.
- Added support for death messages for mobs with nametags and tamed mobs.

DiscordCC V1.0.5
- Fixed an issue whenever the server reloads, the previous message from the specified discord channel get sent to the server again.

DiscordCC V1.0.6
- Added support for discord reply messages will now show on the server.

DiscordCC V1.0.7
- Added support for console (not everything on the console will be logged).
- A message from the specified discord channel can now run a command within the server.

- The system now use the new network feature called "websocket" to listen discord event(s), such as messages.
- Fixed several bugs in API handler, detecting messages from discords, and more.
- Refactored some codes avoiding code repetition.
- New configuration options were added.

- Display 

- Added new slash command that allow you to execute a command in the server. It cannot give you the result of the command, but it will run in the server.
  - `/command <command>`
  - `/deop <player>`
  - `/op <player>`
  - `/allowlist <add/remove/enable/disable> [gamertag]`
  - `/shutdown`
- Fixed issue with discord gateway connection reported by @orwee
- Fixed shutdown notification not sending in the discord channel
